# Müşteri Segmentasyonu (RFM & K-Means)

Bu projede e-ticaret verileri kullanılarak müşteri segmentasyonu yapılmıştır. 
RFM analizi ile müşteri davranışları analiz edilmiş ve K-Means algoritması ile segmentlere ayrılmıştır.

## Kullanılan Teknolojiler
- Python
- Pandas
- NumPy
- Scikit-learn

## Proje Amacı
Müşteri davranışlarını analiz ederek pazarlama stratejilerini geliştirmek ve müşteri sadakatini artırmak.

## Yöntem
- Veri ön işleme
- RFM analizi
- K-Means kümeleme
- Segment analizi
